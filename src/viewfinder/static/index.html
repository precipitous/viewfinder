API Reference

POST /api/ingest

Submit a YouTube URL for processing. Request body (JSON): { "url": "https://youtube.com/watch?v=VIDEO_ID", "prompt": "default", // default | brief | detailed | technical "model": "claude-sonnet-4-20250514", "translate_to": null, // "es", "fr", etc. "transcript_only": false, "backend": "claude", // "claude" or "openai" "base_url": null // for openai backend } Response: video_id, title, channel, word_count, summary, tokens used

GET /api/videos

List all ingested videos with transcript/summary counts.

GET /api/videos/{video_id}

Get full details: metadata, transcript text, all summaries.

GET /api/videos/{video_id}/transcript

Get timestamped transcript snippets.

GET /api/search?q=query

Full-text search across all transcript content.

GET /api/videos/{video_id}/export.md

Download video data as a Markdown file.

GET /api/cost

Cumulative token usage and cost breakdown by model.

GET /api/health

Health check. Returns {"status": "ok", "version": "..."}
